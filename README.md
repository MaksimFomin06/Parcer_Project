# 📘 Проект: "Configurable Book Price Monitor"

> Инструмент для создания парсеров без написания кода. Пользователь настраивает логику сбора данных через веб-интерфейс, а система генерирует конфигурацию и выполняет парсинг.

**Целевой сайт:** `http://books.toscrape.com/` (специально создан для тренировки парсеров)

**Стек:** Python, Django, requests, beautifulsoup4, json, git.

---

## 🚀 Возможности

✅ **Веб-форма** для создания конфигурации парсера (URL, CSS-селекторы, правила фильтрации)   
✅ **Мгновенный предпросмотр** результатов парсинга после сохранения  
✅ **Независимая архитектура**: бизнес-логика отделена от Django, легко тестируется

## ⚙️ Установка

```bash

cd Parcer_Project/

# 2. Создайте виртуальное окружение
```

## 🎯 Использование

### 1. Создание конфигурации

1. Откройте `http://127.0.0.1:8000/config-form/`
2. Заполните форму:
   - **URL**: адрес страницы для парсинга (например, `http://books.toscrape.com/`)
   - **Имя файла**: `my_books.json` (только латиница, цифры, `-`, `_`)
   - **Селекторы**: укажите поле и CSS-селектор для извлечения
     ```
     Поле: title      | CSS: h3 a
     Поле: price      | CSS: .price_color
     Поле: availability | CSS: .instock.availability
     ```
   - **Требования** (опционально): правила фильтрации результатов
     ```
     Поле: price | Оператор: > | Значение: 50
     ```

3. Нажмите **«Отправить»**

### 2. Просмотр результатов

После сохранения конфиг автоматически применяется к странице, и снизу формы отобразятся найденные данные:

```json
[
  {
    "title": "A Light in the Attic",
    "price": "51.77",
    "availability": "In stock"
  },
  ...
]
```

### 3. Файл конфигурации

Сохранённый `config/configs/my_books.json`:

```json
{
  "url": "http://books.toscrape.com/",
  "selectors": {
    "title": "h3 a",
    "price": ".price_color",
    "availability": ".instock.availability"
  },
  "requirements": [
    {
      "field": "price",
      "operator": "gt",
      "value": "50"
    }
  ]
}
```

## 🔧 Формат конфигурации

### Селекторы (`selectors`)

Словарь `{ "имя_поля": "CSS-селектор" }`:

| Поле | Пример селектора | Описание |
|------|-----------------|----------|
| `title` | `h3 a` | Ссылка с названием книги |
| `price` | `.price_color` | Цена в элементе с классом |
| `rating` | `.star-rating` | Рейтинг (звёзды) |

### Требования (`requirements`)

Список объектов с правилами фильтрации:

```json
{
  "field": "price",        // Поле из selectors для проверки
  "operator": "gt",        // Оператор: gt, lt, eq
  "value": "50"            // Значение для сравнения
}
```

**Поддерживаемые операторы:**

| Оператор | Описание | Пример |
|----------|----------|--------|
| `gt` | Больше чем | `price > 50` |
| `lt` | Меньше чем | `price < 100` |
| `eq` | Равно | `availability == "In stock"` |
